About

Student

My name is Raed Yazbik. I built this small front‑end project as practice for an HTML/CSS/JavaScript course, combining a clean layout with interactive features. I’m interested in travel and cultural history, so I picked monuments that represent different regions and time periods.

Project purpose

I chose these monuments because each one highlights a different kind of human achievement — engineering (Eiffel Tower), ancient construction (Giza), sacred space (Kaaba), urban entertainment architecture (Colosseum), large‑scale defense (Great Wall), art and symmetry (Taj Mahal), and a city carved into stone (Petra). The goal was to demonstrate semantic HTML structure, responsive CSS that adapts to smaller screens, and JavaScript interactions such as an active navigation state, a gallery lightbox, and a validated feedback form that stores messages locally.

Features implemented

  • Consistent navigation menu on every page
  • Responsive layout with 2 breakpoints (768px / 480px)
  • Gallery grid with hover effects and lightbox preview
  • Feedback form with client-side validation and success state
  • Active navigation highlight based on current page

Tools used

  • HTML (structure and semantics)
  • CSS (layout, responsiveness, accessibility)
  • JavaScript (interactions + validation)
