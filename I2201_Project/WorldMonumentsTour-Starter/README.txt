World Monuments Tour (Starter Kit)
=================================

This is a starter project using ONLY HTML + CSS + JavaScript.
Open index.html with Live Server (or just open in browser).

What YOU must edit to make it submission-ready:
- Review and personalize the written content (the starter placeholders have been filled with sample text).
- Images are now embedded on monument and gallery pages using Wikimedia Commons 'Special:FilePath' URLs (hotlinked). If your class requires local images, download the files listed below and update the HTML to point to your local copies.
- Double-check your instructor’s exact required structure/word counts.

Features already implemented:
- Menu on every page
- Responsive CSS (2 breakpoints: 768px / 480px)
- Gallery grid + hover effect + lightbox preview
- Feedback form validation + localStorage success message
- Active menu highlight (JS)

Files:
- css/styles.css
- js/main.js
- index.html, gallery.html, about.html, feedback.html
- monument-*.html (7 monuments)

Good luck — and make sure you can explain the JS functions during the final.
